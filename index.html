Summary: Most Popular Characters

The purpose of this project was to analyze the popularity of Marvel Comics characters. Using Marvels API, we gathered comicbook, comicbook series and comicbook event data by character. We used matplotlib to create visualizations to show the relationship between different content marvel creates. This site provides the data and visualizations created and descriptions of each.
